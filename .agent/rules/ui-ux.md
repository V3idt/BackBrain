### 4. UI/UX Aesthetics
- **Premium Feel**: make sure things are modern, premium feeling.
- **Avoid Vibe-Coded Excess**: Premium should come from clarity, restraint, and quality, not decorative noise or trend-chasing.
- **Reuse Existing UI Patterns**: Prefer established components and interaction patterns before creating custom UI.
- **No Emojis in Code or UI**: Keep presentation intentional and professional.
